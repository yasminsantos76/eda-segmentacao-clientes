# EDA – Segmentação de Clientes

Este projeto tem como objetivo realizar uma análise exploratória de dados (EDA) em uma base de clientes com informações demográficas, comportamentais e de consumo, com foco em gerar insights para estratégias de marketing.

## Principais análises
- Distribuição de renda
- Perfil de escolaridade e estado civil
- Relação entre renda e gastos
- Impacto do número de filhos no consumo
- Engajamento digital e comportamento de compra

## Tecnologias utilizadas
- Python
- Pandas
- Seaborn
- Matplotlib
- Google Colab

Projeto desenvolvido como parte do meu portfólio de Análise de Dados.
